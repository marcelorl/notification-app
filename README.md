# Notification App

## Installation
```shell
$ pnpm install
$ docker-compose up --build
```

## Usage

You`ll have 2 ports opened to work:
 - 5173 - frontend
 - 3000 - backend

Please, open http://localhost:5173 and start using the app.

## Tests

```shell
$ pnpm test
```

## Technologies

 - PNPM
 - PNPM workspaces
 - Vite
 - React
 - Typescript
 - Nestjs
 - Docker
 - Mongodb
 - Jest
